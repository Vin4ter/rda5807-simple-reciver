# Многофункциональный радиоприемник на RDA5807 и Arduino

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)
[![Arduino](https://img.shields.io/badge/Arduino-IDE-00979D.svg)](https://www.arduino.cc/)
[![Platform](https://img.shields.io/badge/platform-Arduino%20Uno%2FNano-green.svg)]()

Полнофункциональный программно-аппаратный комплекс радиоприема на базе тюнера **RDA5807** и платформы **Arduino**. Устройство поддерживает прием FM, авиационного (AIR) и любительских диапазонов (6 м), оснащено Bluetooth-модулем для беспроводного аудио, OLED-дисплеем и интуитивным управлением через энкодер.

## 🚀 Возможности

### 📻 Радиоприем
- **FM** (87.5–108 МГц) — стандартный вещательный диапазон
- **AIR** (118–137 МГц) — авиационный диапазон
- **6 м** (50–54 МГц) — любительский VHF-диапазон

### 🎵 Bluetooth-режим
- Переключение на внешний модуль **JDY-62** для воспроизведения аудио со смартфона

### ⚙️ Настройки приемника (сохраняются в EEPROM)
- **AFC** — автоподстройка частоты
- **Bass Boost** — усиление низких частот
- **RDS** — отображение названия радиостанции
- **FM Deemphasis** — 50/75 мкс / выкл.
- **LNA** — настройка входного усилителя (мощность/порт)
- **Noise Blend Filter** — фильтр шумоподавления
- **New Demodulator Method** — альтернативный метод демодуляции
- **Шаг перестройки** — 25, 50, 100, 200 кГц
- **Моно/стерео** — принудительное переключение
- **DX Mode** — режим дальнего приема с пониженным энергопотреблением

### 🔋 Энергосбережение
- Автоматический переход в сон при отсутствии активности
- Мониторинг напряжения питания через делитель
- Защита от низкого напряжения — автоматическое отключение

### 💾 Сохранение состояния
- Все настройки, текущая частота, диапазон и громкость сохраняются в **EEPROM** и восстанавливаются при включении

## 🛠️ Технический стек

| Компонент | Назначение |
|-----------|-----------|
| Arduino (ATmega328P) | Управляющий микроконтроллер |
| RDA5807 | FM/AM-тюнер с поддержкой RDS |
| SSD1306 | OLED-дисплей 128×64 (I²C) |
| JDY-62 | Bluetooth-модуль для аудиопотока |
| Энкодер (CLK, DT, SW) | Управление навигацией и настройками |
| Делитель напряжения (R1, R2) | Мониторинг напряжения питания |

### 📚 Используемые библиотеки
- `Wire.h` — I²C-взаимодействие
- `Adafruit_GFX` / `Adafruit_SSD1306` — работа с OLED-дисплеем
- `GyverEncoder` — обработка энкодера
- `GyverPower` — энергосберегающие режимы
- `RDA5807` — управление тюнером
- `EEPROM.h` — сохранение настроек

## 🔌 Подключение

| Компонент | Вывод Arduino |
|-----------|---------------|
| RDA5807 SDA | A4 (SDA) |
| RDA5807 SCL | A5 (SCL) |
| SSD1306 SDA | A4 (SDA) |
| SSD1306 SCL | A5 (SCL) |
| Энкодер CLK | D2 |
| Энкодер DT | D3 |
| Энкодер SW | D4 |
| Bluetooth JDY-62 | D10 |
| Делитель напряжения (A7) | A7 |

*Подробная схема подключения представлена на фото выше.*

## 🕹️ Управление

| Действие | Результат |
|----------|----------|
| Вращение энкодера | Изменение частоты |
| Вращение с удержанием | Регулировка громкости |
| Короткое нажатие | Переключение диапазона / режима |
| Длинное удержание | Вход в меню настроек |
