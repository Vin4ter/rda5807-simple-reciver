
## 🔧 Установка и прошивка

1. Установите **Arduino IDE** (версия 1.8.13 или новее)
2. Установите необходимые библиотеки через менеджер библиотек:
   - `Adafruit SSD1306`
   - `Adafruit GFX`
   - `GyverEncoder`
   - `GyverPower`
   - `RDA5807` (автор: [matthijskooijman](https://github.com/matthijskooijman/RDA5807))
3. Подключите Arduino к ПК через USB
4. Выберите плату **Arduino Uno / Nano** и соответствующий порт
5. Загрузите скетч `rda_radio_stable05.ino`

> 💡 **Примечание:** При подаче напряжения выше 4.4 В устройство переходит в режим ожидания прошивки (отображается сообщение "Firmware flash").

## 🧪 Тестирование

После прошивки устройство автоматически:
- Загружает последние сохраненные настройки
- Отображает текущую частоту, уровень сигнала (RSSI) и напряжение питания
- Готово к управлению через энкодер
